## Getting Up and Running Locally

- Clone this repository to your local machine:

```bash
git clone https://github.com/RiccardoAD/real-estate
```

- Move to the cloned directory

```bash
cd estate
```

- Install dependencies

```bash
npm install
```

- Start server

```bash
npm run dev
```
